# 🌫️ SolukOS

A lightweight and customizable cybersecurity-focused Termux environment.

## 📌 About

SolukOS is a Termux customization project designed for cybersecurity learning, Linux practice and terminal customization.

It provides a Linux-inspired terminal experience with a custom Zsh environment, theme system and modular installer.

## ✨ Features

- 🐧 Custom Zsh environment
- 🌫️ SolukOS startup banner
- 🎨 Dark gray terminal theme
- 💻 Linux-style terminal prompt
- ⚡ Useful terminal aliases
- 📦 Automated installation system
- 🧩 Modular script structure

## 🚀 Installation

```bash
git clone https://github.com/SolukAtli/SolukOS.git
cd SolukOS
chmod +x install.sh
./install.sh
```
🛠️ Included Components
`install.sh` — Main installation script
`scripts/` — Installation modules
`config/` — Configuration files
`assets/` — Visual assets
`docs/` — Documentation
